# Dev

Make sure you have dependencies installed :

- node (homebrew)
- nodemon (npm)
- pm2 (npm)
- mongo (homebrew)

`npm i` to get the local packages.

To get dev started, in tow terminals :
`npm run dev_server_localDB`
`npm run dev_database`
